Overview

This page provides an overview of the prerequisites for ME-450 Embedded Systems in Robotics.

  • MS in Robotics students have already completed all of these steps.

Install Ubuntu Linux 24.04 LTS

If you are not already running Ubuntu Linux 24.04 LTS, you will need to install it before taking this course.

  1. Backup your system, if the install goes wrong, you want to be able to recover your data.
  2. The Official Ubuntu Tutorial
  3. Follow these Computer Setup Instructions to
    • Setup your computer
    • Install ROS
  4. To take the course, all students are required to have Ubuntu Linux 24.04 LTS installed on their computer (not in a virtual machine) by the first day of class.
